# ASCHII
Convert your Text to ASCII Art <br>
`Currently ASCHII only supports Text.`

# Installation
```
curl -fsSL https://raw.githubusercontent.com/73LIX/ASCHII-ASCII_GENERATOR/main/aschii/install.sh | bash
```

# Usage
1. TUI Mode
```
aschii
```
2. CLI Mode (pass arguments)
```
aschii -m "your_text"
```
